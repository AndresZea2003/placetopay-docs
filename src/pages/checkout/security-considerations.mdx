# Política de seguridad de contenido

Si has implementado una [Content Security Policy](https://developer.mozilla.org/en-US/docs/Web/HTTP/CSP), se debe tener en cuenta los dominios utilizados por Webcheckout para un funcionamiento correcto.

En caso de declarar únicamente un [default-src](https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers/Content-Security-Policy/default-src), tener en cuenta que esta actuará como valor por defecto para todas las demás directivas ausentes.

Dominios utilizados durante el procesamiento de una sesión:

|  Dominios                        | 
|  ------------------------------- |
| https:your-domain.com            |
| https://3dss.placetopay.com/     |     

Esta tabla puede ser actualizada en cualquier momento, por lo tanto, una vez actualizada, se debe agregar a sus Políticas de Seguridad de contenido.
